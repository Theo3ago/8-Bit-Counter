# 8-Bit-Counter
This simple breadboard project was made to help me understand how shift registers work, using a 74HC595 shift register alongside an Arduino Uno.

Resources:

  - large sized breadboard
  - 26 jumper wires
  - 8 10 Kilo Ohm resitors
  - 8 LED's
  - Arduino Uno R3
  - USB Data Cable
  - 74HC595 shift register
